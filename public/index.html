🚀 Как включить 100% аппаратный проброс USB в ядре Windows:

Для полноценного монтирования устройств (чтобы флешки, ключи ЭЦП, HASP, принтеры и сканеры определялись в Диспетчере устройств Windows как физически вставленные), в ОС Windows требуются два официальных компонента:

1️⃣ На отдающем компьютере (Сервер экспорта):

🌐 Скачать .msi с GitHub (dorssel/usbipd-win)

Установите официальную службу usbipd-win. Вы можете скачать инсталлятор с GitHub по кнопке выше или установить автоматически:

⚡ Windows PowerShell от Админа (Альтернатива)
winget install --id dorssel.usbipd-win -e --source winget

2️⃣ На принимающем компьютере (Клиентский драйвер VHCI):

🌐 Скачать .exe с GitHub (vadimgrn/usbip-win2)

Пакет usbipd-win является только сервером. Чтобы принимать и монтировать устройства в Диспетчер устройств Windows, на принимающий ПК нужно установить клиентский драйвер виртуального хост-контроллера VHCI (usbip-win2):

  • Установка с GitHub (Самый надежный способ): Нажмите кнопку «🌐 Скачать .exe с GitHub» выше, скачайте файл со страницы релизов и запустите установку.
  • Альтернативные клиенты: Также поддерживаются клиенты VirtualHere USB Client или USB Network Gate.

⚠️ Важно: После установки драйвера VHCI обязательно перезапустите программу USB-Link Pro — статус справа изменится на «Активен (VHCI Client OK)»!
