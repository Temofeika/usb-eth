💾 Как открыть подключенную флешку или диск в Проводнике (Этот компьютер):

Установленный драйвер usbipd-win является серверной службой экспорта портов. Для обычной работы с файлами на флешках в ОС Windows в 100 раз быстрее и надежнее работает стандартный сетевой доступ (без необходимости ставить низкоуровневые драйверы ядра VHCI):

  1. На ПК с физически вставленной флешкой: откройте Этот компьютер → правой кнопкой мыши на флешку → Свойства → вкладка «Доступ» → «Расширенная настройка общего доступа...» → поставьте галочку «Открыть общий доступ к этой папке».
  2. На принимающем ПК: в верхней адресной строке Проводника введите IP-адрес сервера (например, \\192.168.109.46) и нажмите Enter.
  3. Нажмите правой кнопкой мыши на появившуюся папку флешки → «Подключить сетевой диск...» — флешка навсегда появится в окне «Этот компьютер»!

🚀 Как включить 100% аппаратный проброс в Windows:

Для полноценной интеграции в Диспетчер Устройств Windows (чтобы флешки, ключи ЭЦП, принтеры и веб-камеры работали без ограничений), установите стандартный открытый драйвер usbipd-win.

⚡ Windows PowerShell (Запустить от имени Администратора)
winget install --id dorssel.usbipd-win -e --source winget
1

Скопируйте команду выше и выполните ее в PowerShell на обоих компьютерах.

2

Перезапустите приложение USB-Link Pro (двойной клик на ярлык или npm start).

3

Готово! Теперь при нажатии «Поделиться» устройство будет транслироваться на уровне ядра Windows!
