# ⚡ USB-Link Pro — Проброс USB-устройств по локальной сети

**USB-Link Pro** — это современная программа для совместного использования и проброса USB-устройств по локальной сети (USB over IP / Network USB Share). Она позволяет выбрать USB-устройство на одном компьютере, нажать кнопку **«Поделиться»**, и мгновенно подключить это устройство на другом компьютере через сеть Ethernet или Wi-Fi.

---

## ✨ Возможности и преимущества

1. **⚡ Автоматическое обнаружение в сети (LAN Auto-Discovery)**
   - Вам не нужно вручную вводить IP-адреса и порты! Программа автоматически транслирует UDP-пакеты в локальной сети и находит все компьютеры с запущенным USB-Link Pro.
2. **🖥️ Реальное сканирование Windows PnP**
   - Программа взаимодействует с ядром Windows через PowerShell и WMI, определяя точные названия всех вставленных USB-устройств, их Vendor ID, Product ID и шины подключения.
3. **💎 Премиальный темный интерфейс (Glassmorphism)**
   - Современный дизайн с неоновыми эффектами, плавной анимацией, всплывающими уведомлениями и статусами реального времени без перезагрузки страницы (WebSockets).
4. **🛡️ Поддержка двух режимов работы:**
   - **Режим сетевой эмуляции (Virtual Mode)**: Работает «из коробки» без установки сторонних драйверов. Отлично подходит для демонстрации сети и передачи сигнальных статусов.
   - **Режим аппаратного проброса ядра (Kernel USBIP)**: Полная интеграция со стандартным открытым драйвером `usbipd-win` / `usbip`. В этом режиме удаленная флешка, принтер, ключ ЭЦП или веб-камера появляются в **Диспетчере устройств Windows** как физически вставленные в USB-порт!

## 🚀 Быстрый запуск

### 1. Установка зависимостей и запуск на Компьютере №1 (Сервер / Поделиться)
Откройте терминал в папке с проектом и выполните:
```powershell
npm install
npm start
```
После запуска откроется веб-сервер на порту `4545`. Перейдите в браузере по адресу:
👉 **http://localhost:4545**

Вставьте любое устройство в USB-порт, найдите его в списке во вкладке **«Мои устройства»** и нажмите кнопку **«🚀 Поделиться по сети»**.

### 2. Подключение на Компьютере №2 (Клиент / Подключить)
Запустите программу `npm start` на втором компьютере в этой же сети (Wi-Fi или кабель).
1. Откройте вкладку **«🌐 Сетевые компьютеры»**.
2. Программа автоматически обнаружит Первый Компьютер и покажет список доступных общих USB-устройств.
3. Нажмите кнопку **«🔌 Подключить устройство»**.
4. Устройство будет заблокировано за вашим ПК и подключено!

## ⚙️ Настройка 100% аппаратного проброса на уровне ядра Windows

Если вам необходимо, чтобы Windows распознавала удаленное устройство на аппаратном уровне (для работы специализированного ПО, драйверов принтеров, токенов КриптоПро и т.д.), установите официальный открытый драйвер USBIP:

Откройте **PowerShell от имени Администратора** на обоих компьютерах и выполните команду:
```powershell
winget install --id dorssel.usbipd-win -e --source winget
```

После завершения установки перезапустите приложение `npm start`. Приложение автоматически распознает службу `usbipd-win` и переключится в режим **Kernel USBIP Mode**!

## 📦 Сборка в отдельный EXE файл (Standalone Desktop App)

Программа полностью настроена для упаковки в один независимый портативный файл `.exe` для Windows (не требующий установки Node.js на других компьютерах).

1. Установите зависимости:
   ```powershell
   npm install
   ```
2. Соберите портативный `.exe` файл (через Electron Builder):
   ```powershell
   npm run build-exe
   ```
3. Готовый файл **`USB-Link-Pro-v1.0.0.exe`** появится в папке `dist/`!
   
При двойном клике на `.exe` открывается полноценное графическое окно приложения с интеграцией в системный трей (в правом нижнем углу возле часов), позволяя программе работать в фоне.

## 🛠️ Технологический стек
- **Desktop:** Electron, Electron-Builder, Native Image Tray.
- **Backend:** Node.js, Express, WebSockets (`ws`), UDP Multicast/Broadcast (`dgram`), Child Process (Windows PnP / PowerShell).
- **Frontend:** Vanilla CSS (Glassmorphism Dark Theme, HSL Colors, Micro-animations), HTML5, ES6 JavaScript.
